# Theme Color Palette

## Overview
The Brynex Labs color palette is designed for high-end software engineering aesthetics. It prioritizes contrast, readability, and a premium "engineering" feel.

## Light Theme
| Token | Value (RGB) | Description |
|-------|-------------|-------------|
| Background | `253, 253, 253` | Soft white primary surface. |
| Secondary BG | `247, 247, 247` | Subtle contrast for sections. |
| Card BG | `255, 255, 255` | Pure white for elevated elements. |
| Foreground | `10, 10, 10` | Deep neutral for primary text. |
| Muted | `115, 115, 115` | Soft gray for auxiliary info. |

## Dark Theme (Default)
| Token | Value (RGB) | Description |
|-------|-------------|-------------|
| Background | `5, 5, 5` | Deep black primary surface. |
| Secondary BG | `10, 10, 10` | Subtle contrast for sections. |
| Card BG | `13, 13, 13` | Dark gray for elevated elements. |
| Foreground | `245, 245, 245` | Soft white for primary text. |

## Accent Palette (Shared)
| Name | Hex | Usage |
|------|-----|-------|
| Accent | `#C2410C` | Primary CTA, focus states. |
| Light | `#EA580C` | Hover states, gradients. |
| Lighter | `#F59E0B` | Highlights, glows. |

## Border & Selection
Borders use dynamic alpha values (e.g., `rgba(0, 0, 0, 0.08)` in light mode) to ensure they feel integrated into their respective surfaces.
